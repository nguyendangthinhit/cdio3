# Dự án Web - CDIO 3
## Yêu Cầu Hệ Thống

Trước khi bắt đầu, hãy đảm bảo máy tính của bạn đã cài đặt:
- **Node.js** (Phiên bản LTS khuyến nghị: 18.x hoặc 20.x trở lên)
- **npm** (Đi kèm khi cài đặt Node.js)

## Hướng Dẫn Tải Thư Viện (Cài Đặt)

   ```bash
   cd "e:\cdio3\Web"
   ```
 Chạy lệnh sau để tải toàn bộ thư viện cần thiết:
   ```bash
   npm install
   ```

## Hướng Dẫn Chạy Dự Án

```bash
npm run dev
```
